# sepsis-patient-death-prediction-logistic-regression
Predicting patient death risk from sepsis-related medical factors using Machine Learning (Logistic Regression). The project includes data preprocessing, exploratory data analysis, model training, evaluation, confusion matrix visualization, and prediction performance analysis.
 
Sepsis Patient Death Prediction Using Machine Learning

 Project Overview

This project predicts whether a patient is at risk of death due to sepsis using Logistic Regression.

The model is trained on medical attributes such as:

- Shock
- Malnutrition
- Alcoholism
- Age
- Infection

Target Variable:
- Death (0 = Survived, 1 = Died)
